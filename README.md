## Temática
Álbum virtual del mundial 2026 presentando selecciones con sus respectivos jugadores.

## Descripción
Este sitio web permite visualizar las selecciones participantes del mundial 2026, con la lista de sus jugadores, sus caracteristicas y una imagen.
Los jugadores pertenecen a una sola seleccion.

## Cómo desplegar el sitio

### Requisitos
- XAMPP o similar con Apache y MySQL.
- PHP 8 o superior.

### Pasos para ejecutar el proyecto

1. Clonar o descargar el repositorio dentro de la carpeta htdocs de XAMPP.

2. Iniciar Apache y MySQL desde el panel de XAMPP.

3. Crear una base de datos en phpMyAdmin llamada:

bd_album_mundial

4. Importar el archivo .sql incluido en el proyecto para generar las tablas y los datos iniciales.

5. Abrir el navegador y acceder al proyecto desde localhost utilizando la ruta donde fue ubicado dentro de htdocs.


Usuario administrador:
